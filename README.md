#  Sharlandia — каталог интернет-магазина воздушных шаров

##  Описание проекта
Sharlandia — это веб-приложение каталога интернет-магазина воздушных шаров.
Проект предназначен для демонстрации работы с клиент–серверной архитектурой,
базой данных, REST API и современным frontend-фреймворком.

Приложение позволяет:
- просматривать каталог товаров;
- фильтровать товары по категориям;
- получать данные из базы данных через API.

Проект разрабатывается как учебный, с возможностью дальнейшего расширения
(личные кабинеты, корзина, оформление заказов, админ-панель).

## 🏗 Архитектура проекта

Проект реализован по клиент–серверной архитектуре:

- Frontend (React) отвечает за пользовательский интерфейс и отображение данных
- Backend (PHP) предоставляет API для получения данных из базы данных
- База данных (MySQL) хранит информацию о товарах и категориях

Взаимодействие между frontend и backend осуществляется по HTTP с использованием JSON.

# Установка зависимостей

## Backend

Установить XAMPP
Скопировать папку проекта в:
C:\xampp\htdocs\
Запустить Apache и MySQL
Создать базу данных sharlandia в phpMyAdmin

## Frontend

cd frontend
npm install

---

# Запуск проекта

## Backend
Backend доступен по адресу: http://localhost/sharlandia/backend/routes/api.php

## Frontend
npm start
Frontend будет доступен по адресу: http://localhost:3000

## Сборка проекта (frontend)
npm run build
